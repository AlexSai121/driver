# UI/UX Core Directives

## 1. Aesthetic Identity
- **Vibe:** NotingOS, BraunDesign, MinimalCozy. 
- **Rule:** Never output flat, generic, or "bootstrap-like" designs. Every component must feel tactile and intentional.

## 2. Typography
- **Prohibited:** NEVER use "Open Sans", Arial, or default generic web fonts. 
- **Required:** Use modern, distinct typography (e.g., geometric sans-serifs, custom variable fonts) that fit a sleek, opinionated aesthetic.

## 3. Materials & Color
- **Surface:** Utilize transparent, glassy, and blurry design elements (`backdrop-filter: blur`, subtle translucent borders). 
- **Gradients:** Apply simple, spectral gradients for backgrounds or accents.
- **Palette:** Use "Material Expressive" color combinations—modern, high-contrast, and dynamic, avoiding dull grays.

## 4. Motion & Interaction
- **Animation:** Focus heavily on morph animations (e.g., shared element transitions) and micro-interactions. 
- **Feedback:** Elements must react to hover, tap, and focus states fluidly. No sudden, jarring state changes.
